# dofo 🚀

dofo, sade ve disiplinli bir günlük planlama uygulamasıdır. "Zaman akıp gidiyor, mazeretlere yer yok" felsefesiyle tasarlanan dofo, hedeflerinize ruthlessly (acımasızca) odaklanmanızı sağlar.

## Özellikler
*   📅 **Günlük Planlama:** Günlük işlerinizi sıralayın, sürükleyip bırakarak önceliklendirin.
*   ✏️ **Profil Özelleştirme:** Kolayca isminizi düzenleyin (dinamik çoklu dil destekli).
*   🔔 **Akıllı Hatırlatıcılar:** Önemli hiçbir işi unutmayın, bildirim izin kontrolleriyle güvenli alarm kurun.
*   ⏳ **Geciken Görevler:** Dünden kalan işlerinizi tek dokunuşla bugüne taşıyın.
*   💾 **Tamamen Çevrimdışı (Local Storage):** Verileriniz güvenle sadece cihazınızda saklanır.
*   🌎 **Çoklu Dil Desteği:** Türkçe, İngilizce, Almanca, Fransızca ve İspanyolca desteklenir.

## Kurulum ve Başlangıç

1. Bağımlılıkları yükleyin:
   ```bash
   npm install
   ```

2. Uygulamayı Expo üzerinde başlatın:
   ```bash
   npm run start
   ```

3. Geliştirme cihazınızda test edin:
   * iOS Simulator veya Android Emulator kullanın.
   * Fiziksel cihazınızda **Expo Go** uygulamasını açarak QR kodu taratın.
